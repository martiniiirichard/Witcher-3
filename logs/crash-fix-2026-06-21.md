- Process: `witcher3.exe`
- Path: `bin\x64_dx12\witcher3.exe`
- Exception: `0xc0000005`
- Faulting module: `witcher3.exe`

The active Windows Documents folder is:

```text
C:\Users\marti\OneDrive\Documents
```

The active Witcher config folder was therefore:

```text
C:\Users\marti\OneDrive\Documents\The Witcher 3
```

## Fix Applied

