## Decision

Do not install yet.

## Reason

AMM requires two main files:

1. `AMM 4.04 Scripts and Horse Mod`
2. `AMM 4.0X Menu Inputs and DLCs`

Only the scripts archive was found in Downloads. The `AMM 4.0X Menu Inputs and DLCs` archive is required for menu XML, input bindings, and DLC folders.

## Instructions Read

The AMM page says:

- Install the scripts file to the `mods` folder.
- The `AMM Menu Inputs and DLCs` file requires manual steps and cannot be installed with NMM.
- Copy its `bin` folder into the main Witcher 3 install folder.
- Copy `DLCAMM` and `DLCAMMHORSE` into the game `dlc` folder.
- Paste `inputs.txt` into the beginning of `input.settings`.
- Paste `hiddenxml.txt` into `hidden.xml` above the last `</VisibleVars>`.
- Paste `inputsxml.txt` into `input.xml` below the first `<VisibleVars>`.
- Paste `nextgensteps.txt` at the bottom of both `dx11filelist.txt` and `dx12filelist.txt`.
- Add lines; do not replace/delete existing lines.

## Compatibility Risk

AMM modifies high-conflict scripts:

- `playerWitcher.ws`
- `witcherSword.ws`
- `throwable.ws`
- `combatSword.ws`
- `horseManager.ws`

The W3EE compatibility patch `AMM and W3EE Redux NextGen 1.39b` contains a merged `playerWitcher.ws` and must load above AMM and W3EE/Redux.

## Required Next Step

Download `AMM 4.0X Menu Inputs and DLCs` from the AMM files page before installing anything.
